ava (gen2)
==========

- riot.js html5 application build with rollup
  - cordova build for ios
  - graphql client
- apollo-server, apollo-client graphql query, mutation, subscriptions


## Development

```
yarn html:watch
yarn html:dev
```

```
yarn node:dev
```

```
yarn cordova:run
```
